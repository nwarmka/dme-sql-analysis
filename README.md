# DME Claims Analysis Using SQL

## Project Overview
This project analyzes Durable Medical Equipment (DME) claims data using SQL to identify trends in equipment usage, revenue, and insurance coverage.

## Dataset
The dataset includes:
- Claim ID
- Patient ID
- Equipment type
- Category
- Date
- Cost
- Insurance paid
- Region

## Key Questions
- Which equipment is used the most?
- What is the total revenue?
- How much is paid by insurance vs patients?
- Which region generates the most revenue?

## SQL Skills Used
- SELECT
- GROUP BY
- ORDER BY
- COUNT
- SUM

## Key Findings
- Equipment usage was evenly distributed across categories
- Total revenue was 6150
- Insurance covered ~78% of costs
- The East region generated the highest revenue
